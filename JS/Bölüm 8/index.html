Fonksiyon Nedir?

Fonksiyonlar, belirli bir görevi gerçekleştiren kod bloklarıdır. Fonksiyonlar sayesinde kod tekrarını azaltır ve kodu daha güvenli hale getiririz.

Arrow Function yazım kuralları

const toplaArrow = (a,b)=> a+b;

Tanımlanan fonksiyon içinde yazılacak kod satır sayısı 1 ise süslü parantez kullanmamıza gerek yoktur.

const toplaArrow1 = (a,b)=>{ console.log("toplaArrow1 çalıştı"); return a + b; }

Tanımlanan fonksiyon içinde yazılacak kod satır sayısı 1 den fazla ise süslü parantez kesinlikle kullanmamız gerekir.

const kareAl2 = x => x**2;

Tek 1 parametre varsa parantez kullanmak zorunda değiliz.

const merhabaDe = ()=> "Merhaba";

Eğer tanımlanan fonksiyon parametresiz bir fonksiyon ise () kullanmak zorundayız.
